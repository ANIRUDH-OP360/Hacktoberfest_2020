# Problem 1
Write a program that asks the user for the number of the day in a week and prints which day it is.

### Example:
```enter number: 1
Thats Monday!
another one(y/n)? y
enter number: 5
Thats Friday!
another one(y/n)? i guess?

another one(y/n)? n
Exiting...
```
