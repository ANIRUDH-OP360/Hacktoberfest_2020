# Problem 10
Write a program asking the user for a positive
integer n and then prints out the sum of the first n terms of the series 1<sup>4</sup> + 2<sup>4</sup> + 4<sup>4</sup> + 7<sup>4</sup> + 11<sup>4</sup> +  ....

### Example 1:
```
Enter n: 5
The sum is 17315
```

### Example 2:
```
Enter n: 3
The sum is 273
```

### Example 3:
```
Enter n: 0
Invalid input!
```
